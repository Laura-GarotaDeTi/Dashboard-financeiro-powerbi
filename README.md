# Dashboard-financeiro-powerbi
Nesse projeto trabalhamos com um Dashboard financeiro, focado em responder perguntas importantes para o negócio, como:

📊 A empresa está lucrativa? 
📊 Qual é a margem de lucro? 
📊 Quais são as despesas? 
📊 Quais categorias consomem mais recursos?

Durante o desenvolvimento, passamos por algumas etapas importantes:

🔹 Importação dos dados a partir do Excel 
 
🔹 Tratamento e limpeza dos dados no Power Query 

🔹 Criação do layout no PowerPoint e importação para o Power BI como template.
